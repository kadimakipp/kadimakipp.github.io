安装Package Control

官网传送门

简单方法:

  1. View->Show console
  2. 最下放输入:

但是由于被墙，已输入就死机，所以使用另外一种方法

手动

  1. 点击Preferences -> Browse Packages
  2. 调到上层文件夹 Installed Packages下面
  3. 下载Package Control.sublime-package，并复制到Installed Packages下面
  4. 重启sublime
  5. 如果不行删除多试几次

禁止检查更新

  1. 点击Preferences -> Package Settings -> Package Control -> Settings user
  2. 大括号内添加 "update_check": false 注意要在上调语句后加,
  3. 如果这样还不好使的话，打开/etc/hosts文件， 添加一句127.0.0.1 www.sublimetext.com
  4. 重启生效
